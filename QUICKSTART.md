## Что нужно сделать:

### 1. Получить токен бота

1. Открой Telegram и найди @BotFather
2. Отправь команду: `/newbot`
3. Введи имя бота (например: "My Task Manager")
4. Введи username бота (должен заканчиваться на `bot`, например: `my_tasks_bot`)
5. Скопируй полученный токен (выглядит так: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 2. Вставить токен в .env

Открой файл `.env` и замени строку:
```
BOT_TOKEN=your_bot_token_here
```

на:
```
BOT_TOKEN=твой_токен_от_BotFather
```

### 3. Запустить бота

```bash
python bot.py
```

Если все настроено правильно, увидишь в консоли:
```
INFO - Bot is starting...
INFO - Scheduler started
INFO - Bot started successfully!
```

### 4. Протестировать бота

1. Найди своего бота в Telegram по username
2. Отправь команду `/start`
3. Бот попросит поделиться геолокацией для определения часового пояса
4. После настройки появится главное меню

## Примеры использования:

### Создать простую задачу:
1. Нажми "➕ Новая задача"
2. Введи: "Купить продукты"
3. Введи: "завтра 18:00"
4. Выбери приоритет
5. Выбери "Нет" для повторения
6. Выбери "Нет" для кастомного напоминания

### Создать повторяющуюся задачу:
1. Нажми "➕ Новая задача"
2. Введи: "Тренировка"
3. Введи: "сегодня 19:00"
4. Выбери приоритет
5. Выбери "Ежедневно"
6. Задача будет автоматически создаваться каждый день после выполнения

## Возможные проблемы:

### Бот не отвечает:
- Проверь, что токен правильно вставлен в .env
- Убедись, что бот запущен (python bot.py)
- Проверь логи в файле bot.log

### Ошибка подключения к БД:
- Проверь интернет-соединение
- Убедись, что данные БД в .env правильные

### Напоминания не приходят:
- Убедись, что бот запущен
- Проверь, что время напоминания еще не прошло
- Посмотри логи в bot.log

## Полезные команды:

- `/start` - начать работу / перезапустить бота
- `/help` - справка по командам

## Логи:

Все события записываются в файл `bot.log`
Можно открыть его для отладки проблем.

## Остановка бота:

Нажми `Ctrl+C` в консоли где запущен бот.
