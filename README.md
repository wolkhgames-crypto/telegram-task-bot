# Telegram Task Bot

Telegram-бот для управления задачами с напоминаниями, приоритетами и повторяющимися задачами.

## 🚀 Возможности

- ✅ Создание задач с датой и временем
- 🔔 Автоматические напоминания (настраиваемые шаблоны)
- 🎯 Приоритеты задач (высокий, средний, низкий)
- 🔁 Повторяющиеся задачи (ежедневно, еженедельно, ежемесячно)
- 📅 Фильтр задач на сегодня
- ✅ История выполненных задач
- 🌍 Поддержка часовых поясов
- 📋 Пагинация списка задач

## 📋 Требования

- Python 3.10+
- PostgreSQL база данных
- Telegram Bot Token (получить у @BotFather)

## 🛠️ Установка

### 1. Клонирование репозитория

```bash
cd telegram-task-bot
```

### 2. Создание виртуального окружения

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка базы данных

#### Вариант А: Использование Neon (бесплатный хостинг)

1. Зарегистрируйся на https://neon.tech
2. Создай новый проект
3. Скопируй connection string

#### Вариант Б: Локальная PostgreSQL

1. Установи PostgreSQL
2. Создай базу данных:

```sql
CREATE DATABASE taskbot;
```

### 5. Инициализация таблиц

Выполни SQL скрипт для создания таблиц:

```bash
# Если используешь psql
psql -h your_host -U your_user -d your_database -f init_db.sql

# Или через любой PostgreSQL клиент (DBeaver, pgAdmin и т.д.)
# Просто выполни содержимое файла init_db.sql
```

### 6. Настройка переменных окружения

Отредактируй файл `.env`:

```env
# Telegram Bot Token (получи у @BotFather)
BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz

# PostgreSQL Database
DB_HOST=your_host
DB_PORT=5432
DB_NAME=your_database
DB_USER=your_user
DB_PASSWORD=your_password
DB_SSLMODE=require
```

### 7. Создание бота в Telegram

1. Найди @BotFather в Telegram
2. Отправь команду `/newbot`
3. Следуй инструкциям для создания бота
4. Скопируй полученный токен в `.env` файл

### 8. Настройка команд бота (опционально)

Отправь @BotFather команду `/setcommands` и выбери своего бота, затем отправь:

```
start - Начать работу с ботом
help - Справка по командам
```

## ▶️ Запуск

```bash
python bot.py
```

Бот запустится и будет готов к работе!

## 📖 Использование

### Основные команды

- `/start` - Начать работу с ботом (регистрация)
- `/help` - Справка по командам

### Главное меню

После запуска бота доступны следующие кнопки:

- **➕ Новая задача** - создать новую задачу
- **📋 Мои задачи** - посмотреть все активные задачи
- **📅 Сегодня** - задачи на сегодня
- **✅ Выполненные** - история выполненных задач
- **⚙️ Настройки** - настройки напоминаний и часового пояса
- **❓ Помощь** - справка

### Создание задачи

1. Нажми **➕ Новая задача**
2. Введи название задачи
3. Введи дату и время (примеры: "завтра 18:00", "06.04.2026 18:00")
   - Или выбери быструю кнопку: Сегодня / Завтра / Через неделю
4. Выбери приоритет: 🔴 Высокий / 🟡 Средний / 🟢 Низкий
5. Выбери режим повторения (или "Нет")
6. Добавь кастомное напоминание (опционально)

### Действия с задачами

Для каждой задачи доступны кнопки:

- **✅** - Отметить выполненной
- **✏️** - Редактировать (название, дату, приоритет, повторение, напоминания)
- **🗑️** - Удалить

### Настройки

В разделе **⚙️ Настройки** можно:

- Добавить шаблоны напоминаний (например: "за 2 часа", "за 3 дня")
- Удалить существующие шаблоны
- Изменить часовой пояс

Шаблоны напоминаний автоматически применяются ко всем новым задачам.

## 🔧 Технологии

- **aiogram 3.4.1** - библиотека для Telegram Bot API
- **PostgreSQL** - база данных
- **psycopg2** - драйвер PostgreSQL
- **APScheduler** - планировщик задач для напоминаний
- **dateparser** - парсинг дат на русском языке
- **pytz** - работа с часовыми поясами
- **timezonefinder** - определение часового пояса по геолокации

## 🐛 Решение проблем

### Бот не запускается

1. Проверь, что токен бота правильно указан в `.env`
2. Проверь подключение к базе данных
3. Убедись, что все зависимости установлены: `pip install -r requirements.txt`

### Ошибка подключения к БД

1. Проверь правильность данных в `.env` (host, port, user, password)
2. Убедись, что база данных создана
3. Проверь, что выполнен SQL скрипт `init_db.sql`

### Напоминания не приходят

1. Проверь, что бот запущен и работает
2. Проверь логи в файле `bot.log`
3. Убедись, что время напоминания еще не прошло

### Неправильное время в задачах

1. Проверь свой часовой пояс в настройках бота
2. При необходимости измени его через **⚙️ Настройки** → **🌍 Изменить часовой пояс**

## 📊 Логирование

Все события бота логируются в файл `bot.log` и в консоль.

Уровни логирования:
- **INFO** - обычные события (запуск, создание задач, отправка напоминаний)
- **WARNING** - предупреждения (ошибки парсинга дат)
- **ERROR** - ошибки (проблемы с БД, ошибки отправки сообщений)

## 🚀 Деплой на сервер

### Вариант 1: VPS/Dedicated Server

1. Установи Python 3.10+ на сервер
2. Склонируй проект
3. Установи зависимости
4. Настрой `.env`
5. Запусти бота через `screen` или `tmux`:

```bash
screen -S taskbot
python bot.py
# Нажми Ctrl+A, затем D для отсоединения
```

### Вариант 2: Systemd Service (Linux)

Создай файл `/etc/systemd/system/taskbot.service`:

```ini
[Unit]
Description=Telegram Task Bot
After=network.target

[Service]
Type=simple
User=your_user
WorkingDirectory=/path/to/telegram-task-bot
ExecStart=/path/to/venv/bin/python bot.py
Restart=always

[Install]
WantedBy=multi-user.target
```

Запусти сервис:

```bash
sudo systemctl enable taskbot
sudo systemctl start taskbot
sudo systemctl status taskbot
```

### Вариант 3: Docker (опционально)

Создай `Dockerfile`:

```dockerfile
FROM python:3.10-slim

WORKDIR /app

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

CMD ["python", "bot.py"]
```

Запусти:

```bash
docker build -t taskbot .
docker run -d --name taskbot --env-file .env taskbot
```
