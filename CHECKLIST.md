## Что осталось сделать:

### ⚠️ ВАЖНО: Получить токен бота

1. Открой Telegram
2. Найди @BotFather
3. Отправь: `/newbot`
4. Введи имя: например "Task Manager Bot"
5. Введи username: например "my_task_manager_bot"
6. Скопируй токен (123456789:ABCdefGHIjklMNOpqrsTUVwxyz)

### ⚠️ ВАЖНО: Вставить токен в .env

Открой файл `.env` и замени:
```
BOT_TOKEN=your_bot_token_here
```

на свой токен от BotFather.

### 🚀 Запустить бота

```bash
python bot.py
```

## Проверка работы:

1. Найди своего бота в Telegram
2. Отправь `/start`
3. Поделись геолокацией (или выбери часовой пояс вручную)
4. Увидишь главное меню с кнопками
5. Создай тестовую задачу

## Основные функции бота:

✅ Создание задач с датой и временем
✅ Приоритеты (высокий, средний, низкий)
✅ Повторяющиеся задачи (ежедневно, еженедельно, ежемесячно)
✅ Автоматические напоминания (настраиваемые шаблоны)
✅ Фильтр "Сегодня"
✅ История выполненных задач
✅ Редактирование всех полей задачи
✅ Пагинация (10 задач на странице)
✅ Поддержка часовых поясов
✅ Умное форматирование дат ("Сегодня, 18:00", "Завтра, 10:00")

## Технологии:

- Python 3.12
- aiogram 3.4.1 (Telegram Bot API)
- PostgreSQL (Neon - бесплатный хостинг)
- APScheduler (планировщик напоминаний)
- dateparser (парсинг дат на русском)
- pytz (часовые пояса)
- timezonefinder (определение часового пояса по геолокации)
