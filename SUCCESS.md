## 🎯 Основные функции:

### Создание задачи:
1. Нажми "➕ Новая задача"
2. Введи название (например: "Купить продукты")
3. Введи дату и время (например: "завтра 18:00")
4. Выбери приоритет: 🔴 Высокий / 🟡 Средний / 🟢 Низкий
5. Выбери повторение: Ежедневно / Еженедельно / Ежемесячно / Нет
6. Добавь кастомное напоминание (опционально)

### Просмотр задач:
- **📋 Мои задачи** - все активные задачи
- **📅 Сегодня** - только задачи на сегодня
- **✅ Выполненные** - история выполненных задач

### Действия с задачами:
- **✅** - Отметить выполненной
- **✏️** - Редактировать (название, дату, приоритет, повторение, напоминания)
- **🗑️** - Удалить

### Настройки:
- Добавить/удалить шаблоны напоминаний
- Изменить часовой пояс

## 🎨 Форматирование дат:

Бот умно форматирует даты:
- "Сегодня, 18:00" - для задач на сегодня
- "Завтра, 10:00" - для задач на завтра
- "6 апреля, 14:00" - для остальных дат

## 🔔 Напоминания:

- Проверяются каждую минуту
- Отправляются автоматически по времени
- Можно настроить шаблоны в настройках
- Можно добавить кастомные при создании задачи

## 🔄 Повторяющиеся задачи:

При выполнении повторяющейся задачи:
1. Текущая помечается выполненной
2. Автоматически создается новая с новой датой:
   - Ежедневно: +1 день
   - Еженедельно: +7 дней
   - Ежемесячно: +30 дней
