# Telegram Task Bot

Telegram-бот для управления задачами с напоминаниями и приоритетами

## Основные возможности:
- ✅ Создание задач с датой и временем
- 🔔 Напоминания в 7:00, 14:00, 19:00 и в момент дедлайна
- 🎯 Приоритеты задач (высокий, средний, низкий)
- 🔁 Повторяющиеся задачи (ежедневно, еженедельно, ежемесячно)
- 📅 Фильтр задач на сегодня
- ✅ История выполненных задач
- 🌍 Поддержка часовых поясов
- 📋 Пагинация списка задач

## Технологии:
- Python 3.10
- aiogram 3.4.1
- PostgreSQL
- APScheduler

## Установка:
1. Создайте Telegram бот через @BotFather
2. Получите токен
3. Настройте переменные окружения
4. Запустите: `python bot.py`
